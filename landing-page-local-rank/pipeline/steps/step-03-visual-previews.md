## Tarefa

Criar tres direcoes visuais distintas para a mesma copy e estrutura.

Regras:

- Cada arquivo deve abrir sozinho no navegador.
- Usar HTML, CSS e JS inline quando necessario.
- Manter fidelidade a copy original.
- Inserir a area do formulario usando o shortcode fixo de `form-config.md`.
- Usar branding real da Local Rank.
- Nao usar estetica SaaS generica, gradientes roxos, layouts stock ou placeholders visiveis.

Direcoes sugeridas:

- Style A: escuro premium, gradiente navy para azul, cards glassmorphism.
- Style B: claro editorial, papel/off-white, boxes navy e acentos verdes.
- Style C: hibrido conversion-focused, hero escuro e secoes claras densas.
