# Configuracao do Fluent Forms

## Shortcode Fixo

Usar este shortcode em todos os previews e handoffs ate que seja alterado neste arquivo:

```text
[fluentform id="1"]
```

## Regras

- O shortcode nao deve ser solicitado como input em cada execucao.
- O preview GitHub deve exibir o shortcode dentro de uma area visual de formulario.
- O handoff Elementor deve instruir o implementador a inserir esse shortcode em um widget Shortcode.
- Se o ID real mudar, atualizar apenas este arquivo.

## Classe Recomendada

Container do formulario:

```text
lr-landing-form
```
