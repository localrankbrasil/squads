# Critérios de Qualidade — Branding Local Rank

## Critérios de Auditoria de Marca (1–10 por critério)

### Visual

| Critério | Descrição | Aprovado se |
|----------|-----------|-------------|
| Paleta de Cores | Cores usadas pertencem ao brand kit | Todas as cores são #62bb46, #172259, #1d40ad, #ffffff, #f9fafb ou variações aprovadas |
| Tipografia | Fontes usadas são Merriweather (títulos) e Roboto (corpo) | Nenhuma fonte não aprovada aparece |
| Logo | Logo usada na variação correta para o fundo | Versão branca em fundos escuros, original em fundos claros |
| Contraste | Texto legível sobre o fundo | Ratio WCAG AA: mínimo 4.5:1 para texto normal, 3:1 para texto grande |
| Hierarquia Visual | Leitura clara: título > subtítulo > corpo | Diferença de tamanho entre níveis ≥ 1.5x |
| Dimensões | Tamanhos corretos para a plataforma | Instagram feed: 1080×1440px ou 1080×1080px |

### Verbal

| Critério | Descrição | Aprovado se |
|----------|-----------|-------------|
| Tom de Voz | Consultivo, empático, direto | Nenhum tom agressivo de vendas ou promessas irreais |
| Vocabulário | Usa termos aprovados da marca | Sem "SEO" bruto, "garantido", "últimas vagas" |
| Mensagem Central | Alinha com o posicionamento Google-first | Reforça que o americano compra pelo Google |
| Persona Correta | Tom adequado para o público | Persona 1 (empático), 2 (consultivo), 3 (técnico/ROI), 4 (honesto) |

### Consistência

| Critério | Descrição | Aprovado se |
|----------|-----------|-------------|
| Coerência Interna | Todos os slides/elementos seguem o mesmo sistema | Sem variações de cor ou fonte entre slides |
| Identidade Reconhecível | Sem ver o logo, dá para identificar que é Local Rank | Verde + Navy + Merriweather = identidade clara |

## Limites de Aprovação

| Resultado | Condição |
|-----------|----------|
| **APROVADO** | Score geral ≥ 7/10 E nenhum critério abaixo de 4/10 |
| **APROVADO CONDICIONAL** | Score geral ≥ 7/10 E critério não-crítico entre 4-6/10 |
| **REPROVADO** | Score geral < 7/10 OU qualquer critério abaixo de 4/10 |

### Critérios Críticos (nota < 4 = reprovação automática)
- Paleta de Cores: uso de cores completamente fora da marca
- Logo: ausente em material oficial ou versão errada
- Tom de Voz: promessas falsas, tom agressivo ou não-empático
