# Squads — Local Rank Brasil

Repositório central dos squads operacionais da Local Rank Brasil.

## Estrutura

```
squads/
├── opensquad/          # Core do Opensquad — criador e orquestrador de squads
├── branding-local-rank/  # Squad de branding
└── vendas-local-rank/    # Squad de vendas
```

## Como usar

Os squads são executados via Claude Code com o framework Opensquad.
Use `/opensquad run <nome>` para executar um squad.

> Squads específicos de clientes ficam no repo do próprio cliente.
