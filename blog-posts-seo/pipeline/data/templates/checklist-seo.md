# Checklist SEO - Blog Post

## Antes De Escrever

- [ ] Idioma definido
- [ ] Titulo exato definido
- [ ] Localizacao alvo definida
- [ ] Tema definido
- [ ] Palavra-chave principal definida
- [ ] Intencao de busca clara
- [ ] Publico-alvo definido
- [ ] Link interno principal coletado ou marcado como ausente
- [ ] Links internos adicionais coletados ou marcados como ausentes
- [ ] Link interno da conclusao coletado ou marcado como ausente
- [ ] Informacoes do cliente coletadas

## Estrutura

- [ ] H1 igual ao titulo fornecido
- [ ] Introducao com 3 paragrafos sem heading
- [ ] 4 secoes principais
- [ ] Cada secao tem 1 H2
- [ ] Cada H2 tem 3 H3s
- [ ] Cada H3 tem 3 paragrafos
- [ ] Conclusao tem H2 que nao usa "Conclusion" ou "In Conclusion"
- [ ] Conclusao tem 2 paragrafos

## SEO

- [ ] Palavra-chave principal no titulo
- [ ] Palavra-chave principal aparece na primeira frase quando natural
- [ ] Palavra-chave principal na introducao
- [ ] Palavra-chave principal aparece no H1
- [ ] Palavra-chave principal aparece no SEO title
- [ ] Palavra-chave principal em pelo menos 1 H2
- [ ] Palavra-chave principal em H3s quando natural
- [ ] Palavra-chave principal aparece em 30% a 75% dos H2/H3, sem keyword stuffing
- [ ] Palavra-chave principal no paragrafo final
- [ ] Meta description escrita
- [ ] Meta description tem 120 a 156 caracteres; ideal 150 a 156
- [ ] Meta description usa voz ativa e combina com o conteudo real
- [ ] Meta title tem cerca de 60 caracteres
- [ ] SEO title alinha com H1 e evita repeticao excessiva
- [ ] Slug sugerido
- [ ] Slug em lowercase, com hifens e baseado na keyword
- [ ] Headings seguem hierarquia H1/H2/H3 sem pular niveis
- [ ] Headings descrevem claramente os paragrafos abaixo
- [ ] Links internos incluidos com anchor text exato
- [ ] Nenhuma URL interna aparece mais de uma vez no artigo
- [ ] Link interno principal da introducao nao reaparece no body
- [ ] Links internos adicionais distribuidos pelo body sem concentrar varios links no mesmo paragrafo
- [ ] Links de cluster usam bloco `Related Articles:` no fim de secoes do body principal, quando aplicavel
- [ ] Bloco `Related Articles:` nao aparece na introducao, conclusao ou FAQs
- [ ] Posts antigos do cluster receberam backlink para o novo artigo, quando aplicavel
- [ ] 5 links externos autoritativos incluidos naturalmente
- [ ] Links externos verificados e funcionando
- [ ] Links externos configurados para abrir em nova janela quando houver HTML
- [ ] Links externos incluem `rel="noopener noreferrer"` quando houver HTML
- [ ] Links quebrados substituidos por fontes equivalentes encontradas via Google
- [ ] Termos relacionados distribuidos naturalmente

## Yoast

- [ ] Voz passiva abaixo de 10%; meta interna abaixo de 7%
- [ ] Frases longas abaixo de 25%; meta interna abaixo de 20%
- [ ] Nenhum bloco entre headings passa de 300 palavras
- [ ] Nenhum bloco H3 passa de 280 palavras; ideal entre 220 e 250 palavras
- [ ] Nenhum paragrafo em H3 passa de 100 palavras; ideal entre 60 e 85 palavras
- [ ] Nenhum paragrafo passa de 150 palavras; meta interna abaixo de 120 palavras
- [ ] Nenhuma frase consecutiva comeca com a mesma palavra
- [ ] Nenhum paragrafo tem duas frases longas em sequencia
- [ ] Cada paragrafo comeca com a ideia principal
- [ ] Transition words em pelo menos 30% das frases; meta interna acima de 35%
- [ ] Complex words abaixo de 10%; meta interna abaixo de 8% quando Yoast Premium estiver disponivel

## Formato

- [ ] Sem bullet points no artigo final
- [ ] Sem listas numeradas no artigo final
- [ ] Sem tabelas no artigo final
- [ ] Sem divisorias no artigo final
- [ ] Tom natural e humano
- [ ] Frases proibidas removidas

## Entrega

- [ ] Featured image prompt gerado
- [ ] 3 sugestoes de imagem de apoio geradas
- [ ] 5 FAQs criadas
- [ ] Arquivo final salvo
- [ ] Trocas de links quebrados documentadas, se houver
- [ ] Contexto atualizado
- [ ] Commit feito
- [ ] Push feito
