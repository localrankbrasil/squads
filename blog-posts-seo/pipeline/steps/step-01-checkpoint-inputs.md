# Step 1 — Coleta de Inputs

## Tipo
checkpoint

## Objetivo
Coletar todos os inputs obrigatórios antes de iniciar qualquer etapa do pipeline.

## Instrução para o Usuário

Antes de começar, forneça as seguintes informações em uma única mensagem:

**Obrigatórios:**
- Idioma do post
- Título exato (será o H1)
- Localização alvo (cidade e/ou estado)
- Focus keyword
- Link interno principal para a introdução (URL + anchor text)
- Links internos adicionais para o corpo (URL + anchor text — pode ser mais de um)
- Link interno para a conclusão (URL + anchor text)
- Informações do cliente: tipo de negócio, serviços, público-alvo, tom, diferenciais e posicionamento

**Opcionais (não bloqueiam o início):**
- Links internos adicionais ainda não definidos

## Critério de Aprovação
Todos os campos obrigatórios preenchidos. Se algum faltar, solicitar antes de avançar.
